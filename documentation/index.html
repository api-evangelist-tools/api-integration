API Documentation

This is the API documentation to show how to programmatically get at the data that drives this website. This documentation is driven by an auto-generated OpenAPI from the YAML data store.

This isn't a dynamic API, it is a static API driven by the YAML data store which is managed in a Google Sheet, and publishe as YAML using JavaScript. Then using Liquid I am generating the JSON API, and the OpenAPI that generates the dynamic API documentation.
